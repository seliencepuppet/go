
整数类型变量的声明和打印

```go
package main
import "fmt"

func main() {
    var a int = 123;
    fmt.Printf("variable a = %d\n", a);
}

// 输出结果为:
variable a = 123
```

浮点数类型变量的声明和打印

```go
package main
import "fmt"

func main() {
    var b float64 = 123.123;
    fmt.Println("variable b =", b);
}

// 输出结果为:
variable b = 123.123
```
