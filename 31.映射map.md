
```go
package main

import "fmt"

/*
映射 (Map)

Map是一种哈希表结构, 是一个key/value序列, 每一个key都是不同的, 而且是无序的
在Map中检索数据的时间复杂度是一个常数
*/

func main() {
    ages := map[string]int {
        "Mike":35,
        "John":23,
    }
    fmt.Println(ages)
}
```

第二种创建map对象的函数

```go
package main

import "fmt"

func main() {
    ages1 := make(map[string]int)
    ages1["Mike"] = 20
    ages1["John"] = 12
    fmt.Println(ages1)
    fmt.Println(ages1["Mike"])
}
```
